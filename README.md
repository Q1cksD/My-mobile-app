# Character+

Мобильное приложение для развития полезных поведенческих привычек через:
- выбор зон развития характера,
- умные ежедневные напоминания,
- вечерние чек-ины,
- простую недельную аналитику.

## Стек
- Expo + React Native + TypeScript
- AsyncStorage для локального хранения
- expo-notifications для push-напоминаний

## Запуск

```bash
npm install
npm run start
```

Для веб-режима:

```bash
npm run web
```

## Что реализовано в MVP
1. Выбор направлений развития (спокойствие, дисциплина, доброжелательность, фокус, пунктуальность).
2. Кастомизация цели для каждой выбранной категории.
3. Настройка push-напоминаний (вкл/выкл и частота в день).
4. Вечерний чек-ин с оценкой дня от 1 до 5 и заметкой.
5. Расчёт успешности за последние 7 дней.
